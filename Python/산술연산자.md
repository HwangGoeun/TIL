# 산술연산자
<br>

- 덧셈(+) : 문자열도 덧셈이 가능하다.<br>
`1 + 1`<br>
`'a' + 'b'`

<br>

- 뺄셈(-)<br>
    `1 - 1`

<br>

- 곱셈(*)<br>
    `1 * 1`

<br>

- 나눗셈 (/) : 소수점까지 계산한다.<br>
    `3/5`

<br>

- 거듭제곱 (**)<br>
`3 ** 3`

<br>

- 몫 (//) : 나머지는 제외하고 몫만 반환한다<br>
`10 // 3`

<br>

- 나머지 (%) : 몫은 제외하고 나머지만 반환한다.<br>
`10 % 3`
